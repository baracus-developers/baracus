Baracus - Build Management
  • Assist with product ISO location and retrieval.
    - Avoid layers of web navigation to hidden URLs.
  • Build tree management.
    - Gracefully handle licenses and add-ons.
  • Single command to setup a network install server.
    - Start / Stop needed services (nfs, http, ftp).
    - Add / Remove configuration as needed.
  • Serve TFTP / PXE / Boot / Autoinstall files directly from database.
  • Build host template management.
    - Register builds.
    - Deploy identical rebuilds.
    - Versioning
    - Historical lookup.
  • User configurable hooks.
